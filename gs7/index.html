Topic: Create Axios Instance

What is an Axios Instance?
An Axios instance is a pre-configured version of Axios that you can reuse. It allows you to set a base URL, default headers, interceptors, and other configuration options for a specific API.

Benefits:
  • DRY (Don't Repeat Yourself) - configure once, use everywhere
  • Easy to maintain - change base URL in one place
  • Separate instances for different APIs
  • Custom interceptors per instance
